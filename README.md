## Keys

* Left click  ==> Zoom in
* Right click ==> Zoom out
* u ==> move up
* d ==> move down
* r ==> move right
* l ==> move left


## Try it

* Clone the repo using
```
git clone https://github.com/vnaazleen/fractal-explorer
```

* Compile FractalExplorer.java
```
javac FractalExplorer.java
```

* Run it
```
java FractalExplorer
```
